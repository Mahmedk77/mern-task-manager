# 📝 Full Stack Task Manager App

A full-featured task manager built using the **MERN stack** (MongoDB, Express.js, React, Node.js) with **Tailwind CSS** for a sleek, responsive UI.

---

## 🚀 Tech Stack

### Frontend
- **React.js** – Component-based architecture for building dynamic UIs
- **Tailwind CSS** – Utility-first CSS framework for rapid UI development
- **Axios / Fetch API** – For making HTTP requests to the backend

### Backend
- **Node.js** – JavaScript runtime environment
- **Express.js** – Minimal and flexible Node.js web application framework
- **MongoDB** – NoSQL database for storing tasks
- **Mongoose** – ODM for MongoDB to define schemas and interact with the database

---

## 📦 Features

### ✅ Core Functionalities

- **Create Tasks**  
  Users can create new tasks with a title and description.

- **Read Tasks**  
  Fetch all tasks or filter by status (e.g., pending, completed).

- **Update Tasks**  
  Edit task details or change task status (e.g., mark as completed).

- **Delete Tasks**  
  Remove individual tasks from the list.
